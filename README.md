### Customization

To customize colors, edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #EB0A1E;    /* Toyota Red */
    --secondary-color: #1a1a1a;   /* Dark Gray */
    --accent-color: #ff6b6b;      /* Light Red */
    --neon-red: #ff0000;          /* Neon Effect */
    --neon-blue: #00ffff;         /* Accent Neon */
}
```
